Welcome to the Software Engineering documentation for the **StudyTrack** project.
This documentation follows the **Software Development Life Cycle (SDLC)** and captures the planning, design, implementation, testing and deployment of the application.
The documents are organized to reflect the progression of the project from initial planning to deployment and future maintenance.

# Documentation Workflow

This documentation evolves alongside the project.

For each phase of development:

1. Complete the planning or design work.
2. Update the relevant documentation.
3. Commmit the documentation changes.
4. Implement the feature.
5. Test the implementation.
6. Record important decisions and lessons learned.
